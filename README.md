# Tarot Journal Vision (塔羅日記) 🔮

一個現代化的塔羅牌日記網頁應用程式，讓使用者可以記錄每日的抽牌、撰寫心得，並獲得深度的靈感洞察。

## 功能 (Features)

- **完整偉特塔羅 (Rider-Waite Tarot)**：78 張牌義詳解 (正位/逆位)。
- **雷諾曼卡 (Lenormand)**：36 張牌義詳解。
- **語言切換**：完整的中英文介面切換。
- **AI 解析**：每人每日限用 3 次，保持覺察與節制。

---

### 設定環境變數 (.env)

請在專案根目錄建立 `.env` 檔案，並填入以下資訊：

```env
VITE_SUPABASE_URL="Supabase_URL"
VITE_SUPABASE_PUBLISHABLE_KEY="Supabase_Anon_Key"
VITE_GOOGLE_AI_KEY="Google_Gemini_API_Key"
```

## 🗓️ Roadmap

### 🔜 近期優先 (Priority High)

- [v] **AI 每日指引 (Daily Guidance)**：每日登入時提供一句靈性指引。
- [v] **首頁列表優化**：優化長篇日記在列表頁的顯示方式 (文字截斷或摘要)。
- [v] **Markdown 渲染增強**：支援更豐富的文字格式 (粗體、清單等)。
- [ ] fix把分享卡片，內容還沒有很正確。
- [v] 在日記中加入情緒標籤（如快樂、悲傷、焦慮），並分析這些情緒與抽到的牌卡之間的關聯

### 📊 分析 (Deep Insights)

- [ ] **共時性分析**：偵測特定時間段內反覆出現的牌 (例如：「這週你抽到了 3 次『寶劍三』」)。
- [ ] **能量熱力圖**：以視覺化方式呈現每週/每月的元素分佈 (火/水/風/土)。
- [ ] **關鍵字雲**：分析日記中最常出現的心情關鍵字。

### 🎭 情緒數據分析 (Emotional Data Analysis)

- [ ] **情緒趨勢圖表**：視覺化呈現一段時間內的情緒變化趨勢，包括情緒強度的波動曲線。
- [ ] **情緒與牌卡關聯分析**：分析特定情緒狀態下最常抽到的牌卡，探索情緒與牌卡之間的深層連結。
- [ ] **情緒平衡儀表板**：展示情緒分佈的平衡狀態，幫助使用者了解自己的情緒模式。
- [ ] **情緒預測提醒**：基於歷史數據，預測可能出現情緒低潮的時期，提供預防性的靈性建議。
- [ ] **情緒日記洞察**：AI 分析情緒標籤與日記內容的關聯性，提供個人化的情緒管理建議。
